# 📚 个人学习与实践仓库

欢迎来到我的个人学习与实践仓库！这个工作区专门用于我不断提升在计算机科学基础、数据科学以及高级 AI 智能体框架方面的技能。

## 🗂️ 目录结构

- **`algorithm/`**: 包含各种数据结构与算法的练习代码（如二叉树、图论、排序、滑动窗口等）。用于打磨解决问题的能力及准备技术面试。
- **`python_doc/`**: Python 高级编程概念实践。包含异步编程（`asyncio`）、装饰器、生成器及元类的代码片段与实验。
- **`Joyful-pandas/`**: 针对 `pandas` 库的系统性学习与练习脚本，涵盖从基础数据操作到高级的分组、聚合及时间序列分析。
- **`hello-agents/`**: 一个关于 LLM 智能体开发的综合实战目录。包含循序渐进的章节练习及完整的演示项目，其中包括：
  - **`helloagents-deepresearch/`**: 结合 LLM 与搜索 API（Tavily）的自主深度检索智能体。
  - **`helloagents-trip-planner/`**: 利用工具调用与状态管理机制的行程规划智能体。
- **`动手学深度学习/`**: 基于 PyTorch 框架的深度学习模型实践练习与代码复现。

## 🎯 仓库宗旨
本仓库作为我的代码沙盒，用于将复杂的技术概念拆解为可执行的脚本与微型项目。它记录了我建立扎实工程基础，并同步探索前沿 AI 技术的成长轨迹。
