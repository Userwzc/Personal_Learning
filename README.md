# 📚 Personal Learning Repository

Welcome to my personal learning and practice repository! This workspace is dedicated to my continuous skill development in foundational computer science, data science, and advanced AI agent frameworks.

## 🗂️ Directory Structure

- **`algorithm/`**: Contains my practice codes for various data structures and algorithms (e.g., Binary Trees, Graphs, Sorting, sliding windows). Used for honing my problem-solving skills and preparing for technical interviews.
- **`python_doc/`**: Advanced Python programming concepts in practice. Includes code snippets and experiments on Asynchronous programming (`asyncio`), decorators, generators, and metaclasses.
- **`Joyful-pandas/`**: Systemic learning and practice scripts for the `pandas` library, covering everything from basic data manipulation to advanced grouping, aggregation, and time-series analysis.
- **`hello-agents/`**: A comprehensive hands-on practice directory following the development of LLM Agents. It features step-by-step chapter practices and complete demo projects, including:
  - **`helloagents-deepresearch/`**: An autonomous research agent combining LLM and search APIs (Tavily).
  - **`helloagents-trip-planner/`**: A trip planning agent utilizing tool calling and state management.
- **`动手学深度学习/`**: Practical exercises and implementations of deep learning models, focusing on the PyTorch framework.

## 🎯 Purpose
This repository serves as a code sandbox where I break down complex technical concepts into executable scripts and small projects. It reflects my journey of building a solid engineering foundation alongside exploring cutting-edge AI technologies.
